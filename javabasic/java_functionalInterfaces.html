Java Predefined Functional Interfaces

Package java.util.function provides these predefined functional interfaces

Interface Method Purpose
UnaryOperator<T> T apply(T m) Apply a unary operation to an object of type T and return result of type T.
BinaryOperator<T> T apply(T m, T n) Apply an operation to two objects of type T and return result of type T.
Supplier<T> T get() Return an object of type T
Consumer<T> void accept(T m) Apply an operation on object of type T
Predicate<T> boolean test(T m) Determine if an object of type T fulfills some constraint. Return outcome as boolean value
Function<T,R> R apply(T m) Apply an operation to an object of type T and return the result of type R.
BiConsumer<T,U> void accept(T m, U n) Use when Consumer needs 2 input parameters
BiPredicate<T,U> boolean test(T m, U n) Use when Predicate needs 2 input parameters
BiFunction<T,U,R> R apply(T m, U n) Use when Function needs 2 input parameters
