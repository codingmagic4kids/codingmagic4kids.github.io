Why Coding Magic ?

The idea behind Coding Magic is to Introduce Programming Concepts to Kids and Teens with Fun and Cool Projects. Learning, be it Programming or anything for that matter, happens only when the Student is Interested and Motivated. Coding Magic, Empowers the Students to Code, by Incrementally Building their Coding-Confidence with Bite-sized Hands-on Projects, right from Session1.

What is the right course for me ?

The Basic courses, Java Basic and Python Basic, are for first time programmers. These courses are designed to provide introduction to the Programming language and build confidence with Practical hands-on sessions.

The Intermediate courses, Java Intermediate and Python Intermediate, are comprehensive courses with in-depth Programming Concepts and Language Specific Features. The Intermediate courses are recommended as the next right step for confident programmers, graduating from the Basic courses.

The Advanced courses, Java Advanced and Python Advanced, are recommended for Advanced Coders and Language Enthusiasts.

If you are not sure which category (Basic/Intermediate/Advanced) you fall under, check out the Course Topics for each. If you are confident with the topics covered in, lets say Basic course, you can pick the next one (Intermediate)
