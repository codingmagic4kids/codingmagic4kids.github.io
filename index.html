Why Coding Magic ?

The idea behind Coding Magic is to Introduce Programming Concepts to Kids and Teens with Fun and Cool Projects. Learning, be it Programming or anything for that matter, happens only when the Student is Interested and Motivated. Coding Magic, Empowers the Students to Code, by Incrementally Building their Coding-Confidence with Bite-sized Hands-on Projects, right from Session1.

What are the recommended courses for kids ?

The Basic courses, Java Basic and Python Basic, are comprehensive enough with in-depth Programming Concepts and Language Specific Features. The Basic courses are recommended for Beginner/Intermediate programmers.

The Advanced courses, Java Advanced and Python Advanced are recommended for Advanced Coders and Language Enthusiasts.
